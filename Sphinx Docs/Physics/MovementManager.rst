===============
MovementManager
===============

.. toctree::
   :maxdepth: 4

.. role:: csharp(code)
   :language: csharp

Manages the movement of the :doc:`Sprites <../Sprite/Sprite>`, based on their velocity and gravity. Also checks for position based on this. The user should never need to worry about this class.

.. warning::
	Collision checking is not currently functioning properly. Please implement your own for now.
